## Help

```
npm install || yarn install
```
-----------
run the app : <br/>
```
npm run dev || yarn dev
```
or <br/>
```
npm run json-server || yarn json-server
```
```
npm start || yarn start
```
  
-----------
Testing : <br />
```
npm run test || yarn test
```
